# Nexa AI

A Local AI Desktop Assistant built with Python and Ollama.

## Features

- Local AI Chat (Ollama)
- Open Applications
- Open Websites
- Google Search
- YouTube Search
- Create/Delete Files
- Create/Delete Folders
- List Files
- Open Files
- Screenshot Capture
- CPU Information
- RAM Information
- Disk Information
- Battery Information
- Basic Memory
- Command Parser
- Task Planner

## Technologies

- Python
- Ollama
- SQLite
- PyAutoGUI
- Faster Whisper
- Edge TTS
- Psutil

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/AI-Assistant.git

cd Nexa-AI

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
python app/main.py
```

## Build EXE

```bash
pyinstaller --onefile --name Nexa app/main.py
```
